# Tasks CLI
Simple command line tool to add, complete and delete tasks.

## Usage
```bash
tasks <commands>
```
- commands = [**`add`** \<**_description_**> | [**`complete`** | **`delete`**] \<**_ID_**>]

## Installation
Either run:
```bash
make tasks
```
Or
```bash
go build -o "tasks.exe" "./cmd"
```
